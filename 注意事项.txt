所有算法必须实现以下函数：
def search(graph, coords, start, goals):
    return goal, nodes_expanded, path


在各自算法文件中：
不要读取文件（禁止 parse）
不要 print 输出
不要修改主程序 search.py

必须使用统一数据结构
graph：邻接表（dict）
coords：节点坐标
Node类：统一使用 node.py 中的类
路径回溯：使用 utils.py 中的 reconstruct_path


在扩展节点时：
必须按节点编号从小到大排序
若条件相同，按加入顺序（FIFO）

运行方式：
python search.py <输入文件> <算法名称>

# 运行 GBFS
python search.py PathFinder-test.txt GBFS
